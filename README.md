# myPort

#### 介绍
通用上位机
可自动解析通信协议
可根据协议自动返回命令
可根据解析内容形成曲线
可保存配置
可保存数据

#### 软件架构
基于c# winform
使用第三方UI库 [SunnyUI](https://gitee.com/yhuse/SunnyUI)
使用VS2019 社区版进行开发


#### 安装教程

打开myPort.exe即可
注意SunnyUI.dll和SunnyUI.Common.dll需要和myPort.exe在同级目录下
